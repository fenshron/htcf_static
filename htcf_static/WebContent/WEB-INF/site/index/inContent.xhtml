赢，一定在未来！


——汇通金融（集团）董事长黄平—乔迁演讲稿

今天注定是一个值得全世界都铭记的日子。

因为今天阿里巴巴公司在美国纳斯达克上市了。这不仅仅意味着马云超过李嘉诚成为新的中国首富，而且在一定程度上标志着以互联网为代表的新兴产业对传统行业的超越。

今天也是一个值得全体汇通人铭记的日子。

今天我们搬了新家。从原来的粮食大厦搬到了现在的卓越城，这是全体汇通人三年来共同努力的结果。

“搬到高档的写字楼好不好？”

“当然好！”

因为我们的办公环境变好了，地方更宽敞了，客户来了更舒心了，我们员工的精气神也更足了，这当然是一件值得高兴的事情。

但是今天我也要给为大家降降温。

我们搬到新家，首先来讲，并不能单纯地看作我们事业的成功，至多也只能算是阶段性的成绩。按我本心来讲，我甚至不认为这是阶段性的成绩，而纯粹是为了改善员工办公环境、提高客户体验的举措，

这样就能让员工更好地为我们的客户提供更优质的服务和体验。为什么这么讲？因为大家都知道我们公司虽然说经营了三年，也在开始盈利，但是，我们是经营风险的企业，我们这个行业永远存在风险，稍不注意就会给我们带来损失甚至灾难性的打击，所以我们时刻要包体警惕，小心翼翼的经营，对自己负责，对企业负责，对投资人负责。

孟子说过：“生于忧患死于安乐”。不可否认，整个行业处于高速发展中。这让我们产生很多幻觉，总认为前途一片美好。但是毁灭起来也是很快！所以，才会在今天给大家掏掏心窝子，说说心里话。

“马云自己说的：“今天很困难，明天更困难，后天很美好。”

阿里巴巴的上市已经很美好”了，但是不要忘了他已经走了十五年，汇通才三年，今天的困难还没走完，相信明天会更难！要熬到“后天”，我相信这个时间会超过十五年，甚至是未来的四十七年，这个过程一定很艰辛很痛苦。但是只要我们坚信、坚持“用50年的时间和心态去创业”。

赢，一定在未来！
